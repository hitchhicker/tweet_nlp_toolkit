# tweet_nlp_toolkit
Tweet NLP toolkit
## Installation
```
python3 -m venv .env
source .env/bin/activate
python -m pip install -U pip
pip install tweet_nlp_toolkit
```
## Usage
### Text Parsing
```python
>>> from tweet_nlp_toolkit.prep.text_parser import parse_text
>>> text = parse_text("123 @hello #world www.url.com ðŸ˜° :) abc@gmail.com")
>>> text.tokens
['123', '@hello', '#world', 'www.url.com', 'ðŸ˜°', ':)', 'abc@gmail.com']
>>> text.hashtags
['world']
>>> text.mentions
['@hello']
>>> text.urls
['www.url.com']
>>> text.emojis
['ðŸ˜°']
>>> text.emoticons
[':)']
>>> text.digits
['123']
>>> text.emails
['abc@gmail.com']
```
### Tagging entities
```python
>>> from tweet_nlp_toolkit.prep.text_parser import parse_text
>>> parse_text(
...     "123 @hello #world www.url.com ðŸ˜° :) abc@gmail.com",
...     emojis="tag",
...     hashtags="tag",
...     mentions="tag"
... ).tokens
>>> ['123', '<MENTION>', '<HASHTAG>', 'www.url.com', '<EMOJI>', ':)', 'abc@gmail.com']
```

### Preprocessing
```python
>>> from tweet_nlp_toolkit.prep.text_prep import prep
>>> prep(
        "123 @hello #world www.url.com ðŸ˜° :) abc@gmail.com",
        emojis="demojize",
        mentions="remove",
        hashtags="remove",
        urls="remove",
        digits="tag",
        emails="remove"
... )
>>> '<DIGIT> :anxious_face_with_sweat: :)'
```

`parse_text` and `prep` share the same parameters, `parse_text` returns a instance of `ParsedText`,
`prep` returns the preprocessed string.
